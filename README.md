# Poster-PIL

Philosophy poster project analyzing the "Dead Men Can't Check If You Kept Your Promise" ethical dilemma through deontological, utilitarian, and egoist lenses. Includes survey results with Chart.js visualizations.

## Structure

```
html/
├── philosophy_poster.html   — Main poster page
├── css/
│   └── poster.css           — All styles (CSS custom properties)
└── js/
    └── charts.js            — Chart.js bar/pie chart configs
```
